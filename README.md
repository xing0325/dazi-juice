# 打字爽 · dazi-juice

中文打字练习网页，强反馈、爽感优先，可导入自己喜欢的文章一边打一边背。

**在线体验：** https://xing0325.github.io/dazi-juice/

## 特性

- **Monkeytype 风格** — 光标直接在原文上平滑滑动，无独立输入框
- **中文 IME 友好** — 正确处理拼音输入法的候选词上屏，按已上屏汉字比对（不像很多打字站中文翻车）
- **打对的字锁定** — 一旦打对就不能 backspace 回退；错字仍可改
- **导入自己的文章** — 粘贴纯文本即可，本地保存（localStorage）
- **无标点模式** — 一键把所有标点变空格
- **时限模式** — 30 / 60 / 120 秒倒计时
- **强反馈** — 粒子爆裂、combo 连击、屏幕轻抖、合成音效，按 combo 渐进强度封顶
- **结算页** — S/A/B/C/D 评级 + CPM 速度曲线
- **4 套主题** — 深 / 纸 / 暖 / 绿屏
- **Tab+Enter 快速重打**

## 技术

单文件 `index.html`，零依赖零构建。纯 HTML + Vanilla JS + CSS，Web Audio 合成音效，localStorage 持久化。

## 本地运行

直接双击 `index.html`，或起一个静态服务器：

```bash
python -m http.server 8110
```

然后打开 http://localhost:8110/
